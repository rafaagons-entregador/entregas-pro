# Entreghia v12.2

Correções desta versão:

- Ícone do app recriado conforme a identidade oficial aprovada.
- Logo do cabeçalho substituída pela versão horizontal oficial, sem texto separado no HTML.
- Removida seleção redundante de motorista na tela de Nova Rota.
- O app agora usa automaticamente:
  - Conta padrão = motorista padrão definido nas configurações.
  - Conta B = primeiro motorista diferente do padrão.
- Analítico diário passa a somar pacotes, insucessos e reclamações de todos os motoristas da rota, inclusive quando a rota foi lançada apenas na Conta B.
- Service Worker atualizado para `entreghia-v12-2`.

Teste primeiro no Live Preview antes de publicar no GitHub Pages.
