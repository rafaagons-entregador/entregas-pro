# Painel de Entregas Pro v11.5

Base: v11.4.3.

Novidade:
- Personalização dos cards do Dashboard em Configurações.
- As preferências ficam salvas no Supabase por usuário.
- Se houver falha de conexão, também fica salvo no navegador como backup local.

Antes de publicar:
1. Rode `supabase_v11_5.sql` no SQL Editor do Supabase.
2. Suba todos os arquivos desta versão no GitHub.
3. Limpe o cache do navegador/PWA se necessário.
